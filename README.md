# Yolov9_Gelan_c_Segmenattion
YOLOv9 Instance Segmentation for debris detection in satellite images. Includes training and evaluation scripts run in Google Colab.

This project demonstrates how to train and evaluate a YOLOv9 instance segmentation model for debris detection on custom satellite imagery. The workflow is implemented and tested in Google Colab.

##Features
Preparation and organization of a custom dataset for segmentation

Training YOLOv9 instance segmentation on debris classes

Evaluation with metrics (Precision, Recall, mAP@0.5, mAP@0.5:0.95)

Visualization of training progress and final predictions

Exporting trained weights and results

Getting Started
Prerequisites
Python 3.8+

Google Colab (recommended)

CUDA-enabled GPU (for training acceleration)

Required libraries: PyTorch, YOLOv9, OpenCV, Matplotlib
